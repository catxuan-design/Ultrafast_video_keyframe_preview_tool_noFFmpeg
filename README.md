# 视频关键帧预览工具

## 项目简介

视频关键帧预览工具是一个功能丰富的视频关键帧提取和预览工具，帮助用户快速了解视频内容并找到感兴趣的片段。

## 功能特点

- **批量处理**：支持添加多个视频文件，批量提取关键帧
- **多种宫格布局**：支持3x3、4x4、5x5、6x6、7x7、8x8等多种宫格布局
- **灵活分辨率**：可选320、480、640像素三种输出分辨率
- **拖拽支持**：支持将视频文件直接拖拽到窗口中
- **实时预览**：处理过程中显示实时进度和日志
- **预览管理**：可浏览和管理已生成的关键帧预览图
- **快速操作**：右键支持预览、删除、打开所在文件夹等快捷操作
- **批量删除**：支持批量删除预览图及相关文件

## 使用方法

### 方法一：使用可执行文件（推荐）

1. 双击运行 `视频关键帧预览工具.exe`
2. 界面说明：
   - 顶部：设置输出目录（默认与视频同目录）
   - 中部左侧：宫格布局选择（3x3 到 8x8）
   - 中部右侧：分辨率选择（320/480/640像素）
   - 下方：文件列表区，支持拖拽或点击添加视频
   - 底部：日志显示区
3. 点击 **"开始处理"** 按钮开始提取
4. 处理完成后，点击文件名可预览生成的图片

### 方法二：使用Python脚本

1. 确保已安装Python 3.8+
2. 安装所需依赖：`pip install opencv-python numpy Pillow tkinterdnd2`
3. 运行命令：`python 视频关键帧预览工具.py`

## 依赖项

- Python 3.8+
- OpenCV (cv2)
- NumPy
- Pillow
- TkinterDnD2（用于拖拽功能）

## 输出说明

处理完成后，会在视频所在目录生成关键帧预览图：
- 文件命名格式：`[视频文件名]_[宫格]_[分辨率]p.jpg`
- 例如：`movie_5x5_320p.jpg`
