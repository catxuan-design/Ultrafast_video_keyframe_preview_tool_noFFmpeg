# 视频关键帧预览工具

## 项目简介

视频关键帧预览工具是一个用于提取和预览视频关键帧的实用工具，帮助用户快速了解视频内容并找到感兴趣的片段。

## 功能特点

- 自动提取视频关键帧
- 生成关键帧预览图
- 支持多种视频格式
- 简单易用的界面
- 快速处理速度

## 使用方法

### 方法一：使用可执行文件
1. 双击运行 `视频关键帧预览工具.exe`
2. 在弹出的界面中选择要处理的视频文件
3. 点击 "开始处理" 按钮
4. 处理完成后，关键帧预览图会保存在视频所在目录

### 方法二：使用Python脚本
1. 确保已安装Python 3.x
2. 安装所需依赖（见下方依赖项）
3. 运行命令：`python 视频关键帧预览工具.py`
4. 按照提示操作

## 依赖项

- Python 3.x
- OpenCV (cv2)
- NumPy
- Tkinter (通常Python内置)

## 安装依赖

```bash
pip install opencv-python numpy
```

## 注意事项

- 处理大视频文件可能需要较长时间
- 确保有足够的磁盘空间存储关键帧预览图
- 支持的视频格式取决于OpenCV的支持情况

## 示例

处理完成后，会在视频文件所在目录生成一个包含关键帧的文件夹，命名为 `[视频文件名]_关键帧`。
